markdown常用格式及快捷键

1至5级标题

ctrl + alt + (1~5)

正文粗体（bold-text） ／ 斜体（italic-text）

bold-text : ctrl + b

italic-text: ctrl + i

正文 粗体 或者 斜体 正文**粗体**或者_斜体_

文字引用

ctrl + > (需要shift的帮助)

备注：不建议使用此快捷键，因为正常敲>符号只需要按两个键就够了

内嵌代码

单行代码 command + ’

插入链接

shift + cmd + k
