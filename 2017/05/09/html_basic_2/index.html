1. 块 (div 和 span)

div与span的区别：div是块级元素，span是内联元素

block level element & inline element (块级元素 & 内联元素)
block level element

  • 浏览器显示时，块元素占据全部可用宽度，且以新行来开始和结束
  • 例如：<h1> <p> <div> <ul> <table>

inline element

  • 浏览器显示时，通常不会以新行开始
  • 例如：<img> <a>
