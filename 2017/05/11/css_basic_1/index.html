CSS学习笔记 — 基础篇

Veröffentlicht am 2017-05-11 | in CSS

基本概念

CSS

  • CSS —— Cascading Style Sheets 层叠样式表
  • 样式 —— 定义了如何显示HTML元素
  • 层叠 —— 多个CSS定义可以层叠合一

定义样式的三种方式

内联样式(在HTML元素内部)
内部样式(在<head>标签内部，直接写)
外部样式(在<head>标签内部，写css文件的引用声明)

层叠优先级

内联样式 > 内部样式 > 外部样式 > 浏览器缺省设置

尺寸(px / em / % 的区别)

px (pixels)

  • 像素值
  • px是绝对单位，它的取值是固定的
  • 如果使用px，在IE浏览器中不支持字体缩放

em

  • em是相对单位，它的取值是可以变化的
  • 1em = 当前的或者父元素的font-size
  • 任意浏览器的default font-size=16px，此时1em=16px,即10px=0.625em

% 百分比

  • %是相对值

块级元素与内联元素

display属性

display是css中最重要的用于控制布局的属性

每个元素都有一个默认的display取值（与元素的类型有关）

取值

  • block 块级元素
  • inline 内联元素

  • none 隐藏

  • <div>是块级元素，display:block

width & height属性

  • div的width和height属性缺省值为100%（重新开始一行并尽可能占满整个容器）,且可以设置
  • span的width和height属性缺省值为auto，且不可设置

内部文字

  • div的内部文字如果超过了范围，会溢出
  • span会根据内部文字自动调整width和height

窗口缩放

  • div如果由于窗口缩放导致显示不下，将会被直接截断
  • span将自动换行
