在MacOS上开启／关闭／重启Mysql服务

发表于 2017-05-16 | 分类于 Mysql MacOS

图形界面操作

直接通过System Preferences - MySQL开启关闭Mysql:

Terminal命令行操作

全局命令mysql.server

直接执行全局命令

  • 启动: mysql.server start
  • 关闭: mysql.server stop
  • 重启: mysql.server restart

Problem Issue #2 - mysql.server command not found

如果找不到mysql.server命令:

这时需要在安装目录/usr/local/mysql/support-files/mysql.server中找到mysql.server工具

在安装目录下执行mysql.server工具

  • 启动: sudo /usr/local/mysql/support-files/mysql.server start
  • 关闭: sudo /usr/local/mysql/support-files/mysql.server stop
  • 重启: sudo /usr/local/mysql/support-files/mysql.server restart

    日志文件位于／usr/local/mysql/your_computer_name.local.err

  1. 检查mysql数据目录/usr/local/mysql/data的访问权限

这里需要注意的是，Mac OS系统下，mysql 的用户组和用户名都是_mysql，Linux 是mysql
